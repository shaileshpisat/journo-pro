
Initial Prompt
This is a Journal app which will make it simple to add entries. The interface is similar to Google Search page which has a centered inout box which will allow to enter the details. Each entry has a entry timestamp, action date, tags, folder, amount, amount type (inflow or outflow) and entity. Based on the text entered it can identify the above all. All entries from here will go to Inbox. The entries in Inbox will resurface every week if it has an action date and is past. The entries without Action date can be removed from Inbox if there is a folder added to the entry or by adding a folder to the entry.

Below the input box show 5-6 blocks of recently added entries for the day. Below it will be blocks having today's action date. Below it will be those entries which need to resurfaced in brick format.
